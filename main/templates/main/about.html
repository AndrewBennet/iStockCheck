About iStockcheck:

iStockCheck is a free tool to provide Apple iPhone 8 stock notifications for Apple Stores in the UK.

iStockCheck was developed by Andrew Bennet; it is the successor to a previous incarnation which ran as a native Mac and Windows app. iStockCheck is open source, released under the MIT licence.

iStockCheck is a Django web app, hosted on Python Anywhere. It works by polling the Apple web endpoint which drives the iPhone reservation page.
