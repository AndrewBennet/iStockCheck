Download as .zip Download as .tar.gz View on GitHub

iPhone 7 availability checker

Select which iPhone 7 variants you are hunting for, and the U.K. Apple Stores to search in. iStockcheck runs in the background, checking for new iPhone 7 pickup availability every minute.

Notifications

As soon as there is an available pickup slot, iStockcheck sends a notification. Additionally, leave iStockcheck running and receive notifications on your smartphone, via Pushbullet or Telegram.
